商品の交換について

以下の条件に限り、商品到着後7日以内で交換のご依頼をお受けいたします。

  • 商品内容物（付属品等）に欠品がある場合
  • 購入された商品と別の商品が届いた場合
  • 商品に欠損がある場合

※当店にご連絡を頂いた上で、送料着払いにて製品をお戻しください。

※航空便・赤帽・バイク便等による返品は応じかねます。予めご了承ください。

※如何なる場合も期間を過ぎてからの商品の交換は一切承っておりません。商品到着後は速やかに商品の状態をご確認ください。

商品の返品について

以下の条件に限り、商品到着後7日以内で返品のご依頼をお受けいたします。

  • 商品に欠品があった場合。
  • 商品に不良・欠損があった場合。
  • 購入された商品と別の商品が届いた場合。
  • 上記に該当し秋葉館で交換商品がご用意出来ない場合。

 外装及び梱包に破損が無く商品、付属品などに欠品が無い（商品に欠品がある場合を除く）場合に限り、送料、手数料を含む商品代金をご返金いたします。詳しくは「返金について」をご確認ください。

※ 如何なる場合も期間を過ぎてからの商品の返品は一切承っておりません。商品到着後は速やかに商品の状態をご確認ください。

お客様都合による交換・返品について

以下の条件に限り、商品発送後7日以内で交換のご依頼をお受けいたします。

  • 未開封であること
  • 未使用であること
  • 商品本体、パッケージ、説明書、付属品、緩衝材等、商品に同梱がすべてそろっていること
  • レシート（領収書、納品書等）秋葉館が発行している書類がすべてそろっていること

※開封前にご購入された商品に誤りがないか必ずご確認ください。

※交換に掛かる送料・手数料はお客様ご自身にご負担いただきます。

※如何なる場合も期間を過ぎてからの商品の交換は一切承っておりません。

交換・返品の除外について

以下の条件の場合、交換・返品はお受けできません。

  • 交換・返品期間、初期不良期間を過ぎてからの商品交換のご依頼。
  • すべての新品アップル製品
    アップル製品につきましてはAppleサポートセンターへ直接お問い合わせください。
    Web：Apple サポートへのお問合せ
  • 取り寄せ品（お取り寄せまたは3～5日と表記されている商品）
  • カスタマイズ等特別仕様になった商品
  • ライセンスが伴う商品（ソフトウェア、コンピュータライセンス）
  • 内容物（外装、内装、包装、商品本体、付属品、レシート等）に欠品がある場合
    ※商品に欠品（パーツや付属品等の欠品）がある場合を除く
  • イメージと違うなどお客様都合での交換のご依頼
  • お客様都合による交換・返品で商品を開封されている場合
  • 消耗品（バッテリー、電池、メディア等）
  • 外装（パッケージ）の汚れや軽微な破損
  • 商品化粧箱に伝票が貼られている等化粧箱へのシール添付
  • 外装（パッケージ）に著しい破損（へこみ、破れ、濡れ等）がある場合で、外装（パッケージ）を開封された場合
  • 商品到着後、お客様の責任により破損、故障、汚れ、不備が生じた場合
  • パーツや部品を交換した場合、また商品そのものを改造、調整された場合
  • セット品で割引がある場合の一部商品のみの交換・返品
    ※初期不良を除く。
  • 特価品、処分品、一点物の商品
    ※初期不良を除く。
  • 交換商品がご用意できる場合
    ※ご返金での対応となります。詳しくは「返品について」「返金について」をご確認ください。

交換・返品手続について

  • 商品が到着し次第すぐに状態を確認してください。
    ※外装破損の場合絶対にパッケージを開封しないでください。
  • 商品に欠品（パーツや付属品の欠品）や欠損ががある場合は速やかにメールもしくはお電話にてご連絡ください。交換品がご用意できる場合は交換の対応となります。
    詳しくは「交換について」をご確認ください。
  • 担当スタッフが商品の状態をお伺いし、交換・返品の対象になるかをご確認いたします。
    ※状態の判断がつかない場合は、メールにて写真をお送りいただく場合がございます。
    ※外装破損の場合、運送会社へ配送状況の確認をいたします。
  • 交換・返品の対象になる場合、商品に不備がある等当店都合の場合は着払いで、お客様都合の場合はお客様のご負担でご返送いただきます。
    ※当店負担かお客様負担かは返送いただく前に当店スタッフからお伝えいたします。
  • 当店に商品が届き次第、交換・返品の対応をいたします。
    ※お客様都合による商品交換の場合、着払いにて商品発送をいたします。

※外装破損での交換のご依頼の場合、配送業者へ配送状況の確認をし、正常に配送したと回答があった場合は交換対応は一切お受けできません。

返金について

代金引換/銀行振込コンビニエンスストア支払い

商品購入の際に頂いております商品代金（送料含む）を銀行振込にてご返金いたします。

※お客様都合による返金の場合、送料、お支払い時の手数料、ご返金の際に掛る手数料をご返金代金から差し引かせていただきます。予めご了承ください。


クレジットカード決済

秋葉館都合によるご返金：カード会社への請求取消依頼をいたします。

お客様都合によるご返金：お客様の銀行口座に、直接振込みをさせていただきます。

※同月内でのご返金の場合は、カード会社の請求を止めることでお客様に請求が発生しないようにいたします。

※月を跨いだご返金の場合、一旦カード会社からお客様へ請求が上がり、翌月相殺する形でカード会社から返金処理がなされまます。

※お客様都合による返金の場合は、送料、お支払い時の手数料、ご返金の際に掛る手数料をご返金代金から差し引かせていただきます。予めご了承ください。
